> 파인튜닝된 나만의 AI + 수익 조회 + 유튜브 관리 + 텔레그램 알림 + 대화.

**English**

> Fine-tuned personal AI + revenue tracking + YouTube management + Telegram alerts + chat.
> Reference: Connect AI (ezerai.xyz) — but instructor was a developer; Cravveo builds from real user friction.

## 💡 핵심 전략 | Core Strategy

### 1. 파인튜닝 데이터 소스
수동으로 Q&A 만들지 않음.
**기존 Obsidian 노트 + GitHub 레포** → 자동 추출 → 파인튜닝 데이터

### 2. UI 개발 시점
React/Vue 등 프론트엔드 → **Part 3 (Day 201~300)** 에서 개발.
지금은 기존 툴 사용. UI 없어도 됨.

### 3. 차별점
강사: 개발자 출신 → 처음부터 완성된 UI로 제품 출시
크라베오: 기술 초보 → 100일 유저로 살면서 진짜 불편함 경험 → 그걸 바탕으로 개발
→ 오히려 더 현실적인 제품 나올 수 있음
