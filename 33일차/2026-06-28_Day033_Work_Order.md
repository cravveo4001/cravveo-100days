## ✅ 단계별 실행 | Step-by-Step

### Step 1 — Ollama Python 라이브러리 설치 | Install Ollama Python Library

Day 002에서 만든 가상환경을 활성화한 후 설치합니다.
Activate the virtual environment created in Day 002, then install.

```bash
# 1. 프로젝트 폴더로 이동 | Move to project folder
cd ~/projects/cravveo_ai

# 2. 가상환경 활성화 | Activate virtual environment
source venv/bin/activate

# 3. (venv) 표시가 나오면 정상 | (venv) prefix means it's active
# 4. Ollama 라이브러리 설치 | Install Ollama library
pip install ollama

# 5. 설치 확인 | Verify installation
python3 -c "import ollama; print('✅ ollama 설치 완료')"
```

> ⚠️ `pip install ollama`를 가상환경 없이 실행하면
> `error: externally-managed-environment` 오류가 납니다.
> 반드시 `source venv/bin/activate` 먼저 실행하세요.
>
> ⚠️ Running `pip install ollama` without venv will cause
> `error: externally-managed-environment`. Always activate venv first.

### Step 4 — 크라베오 AI 함수 만들기 | Create Cravveo AI Function

```python
# cravveo_ai.py
import ollama

def ask_cravveo(question):
    """크라베오 AI에게 질문하는 함수 | Function to ask Cravveo AI"""
    response = ollama.chat(
        model='cravveo',
        messages=[{'role': 'user', 'content': question}]
    )
    return response['message']['content']

# 테스트 | Test
if __name__ == "__main__":
    answer = ask_cravveo("크라베오 컴퍼니가 뭐야?")
    print(answer)
```

이 함수를 다른 스크립트에서 `from cravveo_ai import ask_cravveo`로 가져다 쓸 수 있습니다.
This function can be imported in other scripts with `from cravveo_ai import ask_cravveo`.

### Step 5 — Day 010 브리핑 스크립트와 연결 테스트 | Connect to Day 010 Briefing Script

```python
# cravveo_briefing_test.py
import ollama
from datetime import datetime

today = datetime.now().strftime("%Y-%m-%d")

prompt = f"""오늘은 {today}입니다.
크라베오 컴퍼니의 AI 어시스턴트로서 오늘의 아침 브리핑을 작성해주세요.
다음 항목을 포함해주세요:
1. 오늘의 날짜
2. 크라베오 프로젝트 현재 진행 상황
3. 오늘 해야 할 일 제안
"""

response = ollama.chat(
    model='cravveo',
    messages=[{'role': 'user', 'content': prompt}]
)

print("=" * 50)
print(f"크라베오 AI 아침 브리핑 — {today}")
print("=" * 50)
print(response['message']['content'])
```

크라베오 AI가 아침 브리핑을 생성하면 **자동화 연결의 첫 단계 완성**!

## 🔧 트러블슈팅 | Troubleshooting

**`ollama` 패키지 설치 오류 | Package install error**
→ `pip install ollama --upgrade` 시도 | Try upgrade
→ venv 환경에서 설치 확인 | Check venv activation

**연결 오류 (Connection refused) | Connection error**
→ Ollama 서버가 실행 중인지 확인 | Check if Ollama is running
→ `ollama serve` 또는 `ollama list`로 확인 | Verify with these commands

**답변이 "Unknown"일 때 | When answer is "Unknown"**
→ Day 031 교훈: Modelfile TEMPLATE 형식 확인 | Check TEMPLATE format
→ `ollama run cravveo`로 먼저 테스트 | Test in terminal first

**답변이 너무 느릴 때 | When response is slow**
→ 3B q8_0 모델은 CPU에서 10~30초 걸림 | 3B q8 takes 10-30s on CPU
→ 정상임, 기다리면 됨 | This is normal, just wait
