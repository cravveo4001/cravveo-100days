## 오늘의 목표 | Today's Goal

Day 041에서 "쇼핑몰"/"Gemma-2B" 두 가지 오류가 사라진 건 확인했지만, 딱 그 두 개만 테스트했다. 오늘은 범위를 넓혀서:

1. 학습 데이터 20개에 있는 질문을 **그대로** 물어봤을 때 정확한지 (암기 확인)
2. 학습 데이터에 없는 **다른 표현으로 바꿔 물어봤을 때**도 정확한지 (일반화 확인 — 예: "크라베오가 뭐 하는 곳이야?"처럼 원래 문장과 다르게 물어보기)
3. 답변이 한 번에 끝나는지, 아니면 Day 040/041에서 봤던 것처럼 계속 이어붙이는지

**중요**: 반드시 RAG 없는 워크스페이스(또는 터미널 `ollama run`)로 테스트한다. Day 041에서 RAG가 섞여서 헷갈렸던 실수를 반복하지 않는다.

## 작업 순서 | Steps

### Step 1. 테스트 질문 세트 준비 (10개)
- 학습 데이터 그대로 5개 (예: "크라베오 컴퍼니가 뭐야?", "ablescan.app은 뭐야?")
- 표현을 바꾼 질문 5개 (예: "크라베오가 뭐 하는 곳이야?", "웹 접근성 검사 도구 이름이 뭐야?")

### Step 2. RAG 없는 환경에서 v5 테스트
- "default" 워크스페이스(또는 터미널 `ollama run cravveo:v5`)에서 10개 질문
- 각 답변에 대해: 정확함(O) / 부분 정확(△) / 틀림(X) / 헛소리로 이어짐(계속) 표시

### Step 3. v4와 비교
- 같은 10개 질문을 v4로도 테스트 (있다면 빠르게)
- v5가 v4보다 나아졌는지, 아니면 비슷한지 판단

### Step 4. 결과 정리
- Before/After 비교표 작성
- 남은 문제가 있으면 불편함_기록.md에 추가 (단, Day 040에서 정한 대로 지금 당장 더 고치려 하지 않음)
