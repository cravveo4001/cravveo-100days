## 오늘의 목표 | Today's Goal

Day 042에서 만든 [[../정체성_평가세트|정체성 평가세트]](30문항)로 **v4를 처음부터 끝까지 테스트해서 기준선을 만든다.** 지금까지 v4는 "그럭저럭 안정적"이라고만 알고 있었지, 30문항 전체로 정확히 채점한 적은 없었다. 이 기준선이 있어야 다음에 v7을 만들었을 때 "진짜로 나아졌는지"를 숫자로 비교할 수 있다.

오늘은 재학습도, 데이터 수정도 하지 않는다. **순수하게 지금 있는 v4를 측정만 한다.**

## 작업 순서 | Steps

### Step 1. RAG 없는 환경 준비
- AnythingLLM "default" 워크스페이스 모델을 `cravveo:v4`로 설정 (또는 터미널 `ollama run cravveo:v4`)

### Step 2. 30문항 전체 테스트
- `정체성_평가세트.md`의 A1~A20, B1~B10을 순서대로 질문
- 각 답변을 정답 기준과 비교해서 O/△/X 채점

### Step 3. 결과 기록
- `정체성_평가세트.md`의 "결과 기록표"에 v4 열 채우기
- O/△/X 개수 합계 내기 (예: O 18개, △ 5개, X 7개)

### Step 4. 패턴 분석
- X(틀림)가 많은 질문 유형이 있는지 확인 (예: A열보다 B열에서 더 많이 틀리면 "일반화가 약하다"는 뜻)
- 다음 재학습 때 어디를 개선해야 할지 메모
