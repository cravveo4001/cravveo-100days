# 2026-05-27 Day 004 Log

이 기록은 2026-05-27, Day 004에 해당합니다. 다른 날짜나 다른 Day 번호를 찾고 있다면 이 문서가 아닙니다.

## 오늘의 목표

Ollama를 설치하고 로컬 컴퓨터에서 AI 언어모델을 처음으로 직접 실행한다.

## 실행한 명령어

```bash
curl -fsSL https://ollama.com/install.sh | sh
ollama --version
ollama list
ollama run gemma4:e2b
ollama list
```

## 확인한 결과

- Ollama 설치 완료: ollama version 0.24.0
- AMD GPU 인식 확인: AMD GPU ready
- 모델 다운로드: gemma4:e2b
- AI 첫 대화: 터미널에서 한국어로 대화 성공
- ollama list에서 gemma4:e2b 확인 완료

## 막힌 지점

- 처음에 gemma4 (9.6GB 풀 버전)를 다운받기 시작했으나 용량이 너무 큰 것 같아 변경
- gemma4:2b 태그가 없어서 오류 발생

## 해결 방법

- gemma4:e2b (edge 2B, 경량 버전)로 전환 → 정상 실행 확인
- gemma4:e2b는 모바일/엣지 기기에서도 돌아갈 만큼 가벼운 버전

## 성공 조건
