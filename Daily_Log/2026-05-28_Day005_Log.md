# 2026-05-28 Day 005 Log

## 오늘의 목표

Ollama에서 두 모델을 비교하고, 프롬프트를 바꿔가며 AI 답변이 어떻게 달라지는지 실험한다.

## 실행한 명령어

```bash
ollama list
ollama run qwen3.5:2b
ollama run gemma4:e2b
ollama list
```

## 확인한 결과

- gemma4:e2b vs qwen3.5:2b 같은 질문 비교 완료
- 프롬프트 실험 3가지 진행 (짧은 질문 → 역할 부여 → 조건 추가)
- 프롬프트가 달라지면 답변의 질이 완전히 달라지는 것 직접 확인

## 막힌 지점

- `llama3.2` (태그 없이) 실행 시 메모리 오류 발생
  - `Error: model requires more system memory (24.3 GiB) than is available (8.2 GiB)`
- qwen3.5:2b 답변 생성 중 중간에 멈추는 현상 발생

## 해결 방법

- `llama3.2` 대신 `qwen3.5:2b` 로 교체 (8GB RAM 환경에 적합)
- 멈췄을 때 `Ctrl+C` 로 중단 후 재실행

## 성공 조건

- [x] `ollama list` 에서 설치 모델 확인
- [x] `qwen3.5:2b` 모델 다운로드 완료
- [x] 두 모델에 같은 질문 → 답변 차이 확인
- [x] 프롬프트 3가지 실험 완료

## 오늘의 핵심 발견

같은 질문이라도 역할을 부여하고 조건을 추가하면 답변의 질이 완전히 달라진다.
"어떻게 물어보느냐"가 AI 답변 품질을 결정하는 핵심 요소임을 직접 경험했다.
