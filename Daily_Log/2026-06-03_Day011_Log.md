# 2026-06-03 Day 011 Log

## 오늘의 목표

AnythingLLM 리눅스 앱을 설치하고 실행한다. 오프라인 RAG 챕터 시작.

## 실행한 명령어

```bash
chmod +x AnythingLLMDesktop.AppImage
./AnythingLLMDesktop.AppImage
```

## 확인한 결과

- AnythingLLM .AppImage 다운로드 및 설치 완료
- AppArmor 프로필 자동 생성 (y 선택)
- 첫 실행 → qwen3 모델 자동 연결 확인
- GUI 화면에서 "안녕" 입력 → "안녕하세요! 어떻게 도움이 필요하신가요?" 답변 확인
- RAG 개념 공부 완료

## 막힌 지점

- 에이전트 모드로 진입 (안녕 입력 시 @agent 메시지 출력)

## 해결 방법

- `/exit` 입력으로 에이전트 모드 종료 후 정상 대화

## 성공 조건

- [x] AnythingLLM .AppImage 파일 다운로드 완료
- [x] `chmod +x` 실행 권한 부여 완료
- [x] `./AnythingLLM*.AppImage` 실행 → 앱 화면 열림 확인
- [x] 앱이 정상적으로 표시되는 것 확인

## 오늘의 핵심 발견

- 터미널 코드 없이 클릭만으로 로컬 AI와 대화 가능 — GUI의 힘
- AnythingLLM은 Ollama 모델을 자동으로 감지해서 연결해줌
- RAG = 매뉴얼 들고 있는 신입 직원 — 내 문서를 AI에게 먹이면 그걸 기반으로 답함
