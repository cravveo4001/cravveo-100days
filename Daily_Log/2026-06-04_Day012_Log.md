# 2026-06-04 Day 012 Log

## 오늘의 목표

AnythingLLM에서 Ollama를 직접 연결하고, 워크스페이스를 만들어 로컬 AI와 GUI로 첫 대화를 한다.

## 실행한 작업

- `ollama list` 로 Ollama 실행 상태 확인
- AnythingLLM 실행
- Settings → AI Provider → Ollama 선택
- Base URL `http://127.0.0.1:11434` 입력 후 Save
- 모델 드롭다운에서 qwen3.5:2b 선택
- `cravveo-test` 워크스페이스 생성
- 워크스페이스에서 "안녕. 지금 어떤 AI 모델이야?" 입력 → AI가 Qwen3.5라고 답변
- YouTube 업로드 문서 작성

## 확인한 결과

- Ollama 모델 목록 정상 출력 확인
- AnythingLLM Settings에서 Ollama 연결 성공 — 모델 목록 자동 로딩
- cravveo-test 워크스페이스 생성 완료
- 워크스페이스에서 한국어 첫 대화 성공 — AI가 모델명 포함 답변

- AnythingLLM은 Ollama 연결 후 설치된 모델 목록을 자동으로 가져온다
- 코드 없이 GUI 클릭 몇 번으로 로컬 AI와 대화할 수 있다
