# 2026-06-05 Day 013 Log

## 오늘의 목표

임베딩 모델을 설치하고, 내 문서를 AnythingLLM에 업로드해서 RAG를 직접 실험한다. 문서 없는 AI와 문서 있는 AI의 답변 차이를 Before/After로 확인한다.

## 실행한 작업

- `ollama pull nomic-embed-text` 임베딩 모델 설치
- AnythingLLM Settings → Embedding → Ollama + nomic-embed-text 연결
- `cravveo_intro.txt` 크라베오 소개 문서 직접 작성
- cravveo-test 워크스페이스에 문서 업로드 + 임베딩 처리
- Before 테스트: 문서 없는 워크스페이스에서 "크라베오 컴퍼니가 뭐야?" 질문
- After 테스트: 문서 있는 워크스페이스에서 같은 질문
- YouTube 업로드 문서 작성

## 확인한 결과

- nomic-embed-text 설치 완료 (`ollama list`에서 확인)
- AnythingLLM 임베딩 설정 연결 성공
- Before: "크라베오 컴퍼니가 뭐야?" → AI가 "스폰지밥 크래비 패티 버거 브랜드"라고 엉뚱한 답
- After: 같은 질문 → "1인 기업가가 운영하는 AI 자동화 프로젝트"라고 정확한 답변
- RAG 작동 성공 — 문서 하나로 AI의 답이 완전히 달라짐

## 오늘의 핵심 발견

- RAG는 진짜로 작동한다 — 문서 하나 넣었더니 AI가 내 회사를 알게 됐다
- Before/After 차이가 극적이다. 스폰지밥 버거 → 1인 기업 AI 프로젝트
- 임베딩 모델은 문서를 "기억하는" 역할이다. 대화 모델과 임베딩 모델이 같이 일한다
