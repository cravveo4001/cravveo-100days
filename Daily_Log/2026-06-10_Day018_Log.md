- `!pip install unsloth` 실행 → Unsloth 설치

- GPU 메모리 사용량 확인

## 확인한 결과

- Unsloth 설치 완료 — 버전 출력 확인

- GPU 메모리: 15GB 중 4~6GB 사용 — 4비트 압축 덕분에 여유 있음

1. **ImportError: Unsloth import 실패**
