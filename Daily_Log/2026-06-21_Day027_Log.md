# 2026-06-21 Day 027 Log

## 오늘의 목표

크라베오 전용 데이터를 대폭 확장하고, 학습량을 늘려서 "학습 전 모델"과 "학습 후 모델"의 답변 차이를 눈으로 확인한다.

## 실행한 작업

- 크라베오 전용 데이터 50개 이상 추가 작성 (29개 → 83개)
- HuggingFace 데이터셋 업데이트
- Before 테스트: 학습 전 모델에게 질문 → 엉뚱한 답 기록
- 30 epoch + learning_rate 5e-5로 학습 실행
- After 테스트: 학습 후 모델에게 같은 질문 → 정확한 크라베오 답변
- YouTube 업로드 문서 작성

## 확인한 결과

- 데이터 83개로 확장 완료
- 학습 결과: loss 3.95 → 0.08 (30 epoch)
- **Before/After 비교:**
  - "크라베오 컴퍼니가 뭐야?"
    - Before: "유대인 종목... 주식 재단..." (엉뚱한 답)
    - After: "IT 프로그램 개발, 인터넷 쇼핑몰, 유튜브 채널 등을 혼자서 운영하면서 수익을 창출하는 1인 기업이야." (정확!)
  - "크라베오의 목표가 뭐야?"
    - Before: "전재가의 성과를 달성하여..." (의미 없는 답)
    - After: "1인 스타트업으로 살아남는 거야. 그게 지금 목표야."
  - "Build in Public이 뭐야?"
    - Before: "사람들의 삶을 개선하고자..." (틀린 답)
    - After: "배우는 과정을 숨기지 않고 공개하면서 진행하는 방식이야. 실패도 에러도 다 공개하는 거야."
