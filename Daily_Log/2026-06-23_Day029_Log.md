# 2026-06-23 Day 029 Log

## 오늘의 목표

학습된 모델을 GGUF 포맷으로 변환하고, 로컬 Ollama에 cravveo 모델로 등록해서 내 컴퓨터에서 돌린다. Phase 1(로컬 환경)과 Phase 2(파인튜닝)가 합쳐지는 날.

## 실행한 작업

- 코랩에서 83개 데이터, 30 epoch 학습 실행
- After 답변 정확한지 확인
- `model.save_pretrained_gguf("cravveo-gguf", tokenizer, quantization_method="q4_k_m")` GGUF 변환
- Google Drive로 GGUF 파일 전송
- 로컬 노트북으로 GGUF 파일 다운로드
- Modelfile 작성 (FROM, PARAMETER, SYSTEM 설정)
- `ollama create cravveo -f Modelfile` 모델 등록
- `ollama run cravveo "크라베오 컴퍼니가 뭐야?"` 로컬 테스트
- YouTube 업로드 문서 작성

## 확인한 결과

- GGUF 변환 완료 (q4_k_m, 771MB)
- Ollama에 cravveo 모델 등록 성공
- 로컬 테스트 결과:
  - "크라베오 컴퍼니가 뭐야?" → "한국의 IT 기업이야. 파인튜닝을 배우고 있는 AI 어시스턴트야."
  - 핵심 정보(IT 기업, 파인튜닝)는 맞지만 일부 텍스트 깨짐
- 완벽하지 않은 이유: 1B 모델 + 4비트 양자화 + 83개 데이터의 한계
